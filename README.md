## 📌 Project Overview
Traditional hotel booking processes often suffer from errors and manual workloads. This project develops a centralized **Hotel Reservation Management System (HRMS)** to automate operations, manage inventory, and provide a seamless booking experience for guests.

## 🏗️ Architectural Design
The system follows a **Layered Architecture** approach to ensure:
- **Scalability:** Easy to expand and add new features.
- **Security:** Structured information flow with secure payment integration.
- **Maintainability:** Clear separation between business logic, data, and user interface.

## 🚀 Key Features
### For Guests:
- **Online Booking:** Search, view, and reserve rooms in real-time.
- **Secure Payments:** Integrated payment gateway for safe transactions.
- **Reward Points:** System to redeem points for loyal customers.

### For Admin & Staff:
- **Inventory Management:** Real-time tracking of room availability and housekeeping status.
- **Reporting:** Centralized dashboard for generating operational reports.
- **Promotion Management:** Tools to create and manage hotel offers.

## 🛠️ Software Engineering Artifacts
This repository contains the full engineering lifecycle of the system, including:
- **Software Requirements Specification (SRS):** Detailed functional and non-functional requirements.
- **System Design:** Architectural diagrams and UML modeling.
- **Implementation Strategy:** Structured flow for hotel operations.

## 📁 Repository Structure
